Agregar productos a un mercado en Java
Para agregar productos a un mercado en Java, puedes seguir los siguientes pasos:

Define una clase Producto que represente los productos que se agregarán al mercado. Esta clase puede tener atributos como nombre, precio, cantidad, etc.
public class Producto {
    private String nombre;
    private double precio;
    private int cantidad;

    // Constructor, getters y setters
}
Crea una clase Mercado que se encargue de gestionar los productos. Esta clase puede tener un array o una lista para almacenar los productos agregados.
import java.util.ArrayList;
import java.util.List;

public class Mercado {
    private List<Producto> productos;

    public Mercado() {
        productos = new ArrayList<>();
    }

    public void agregarProducto(Producto producto) {
        productos.add(producto);
    }

    // Otros métodos para gestionar los productos del mercado
}
3 . En tu programa principal, puedes crear una instancia de la clase Mercado y utilizar el método agregarProducto para agregar productos al mercado.

public class Main {
    public static void main(String[] args) {
        Mercado mercado = new Mercado();

        Producto producto1 = new Producto("Manzana", 1.99, 10);
        Producto producto2 = new Producto("Plátano", 0.99, 15);

        mercado.agregarProducto(producto1);
        mercado.agregarProducto(producto2);

        // Otros procesos o interacciones con el mercado
    }
}
En este ejemplo, se crean dos productos (producto1 y producto2) y se agregan al mercado utilizando el método agregarProducto de la clase Mercado.

Recuerda que estos son solo ejemplos básicos y puedes adaptar el código según tus necesidades específicas. Además, es posible que necesites implementar otros métodos en la clase Mercado para realizar operaciones como mostrar los productos, buscar productos, eliminar productos, etc.
